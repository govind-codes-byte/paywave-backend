# PayWave Backend 💳

FastAPI-based UPI wallet system with:

- JWT Authentication
- UPI ID transfer
- UPI PIN verification
- Request money feature
- QR code generation
- Daily transaction limits
- MongoDB integration

## Run Locally

1. Create virtual environment
2. pip install -r requirements.txt
3. Create .env file
4. python -m uvicorn app.main:app --reload
